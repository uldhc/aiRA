Par aiRA

aiRA rīku ir izveidojis Latvijas Universitātes Digitālo humanitāro zinātņu centrs projekta ȬPEN ietvaros (Nr. ZDA-LIP 2025/2). Projekts veicina zināšanu pieejamību un sabiedrības iesaisti zināšanu radīšanā. Šis rīks ir atvērtā koda projekts un pieejams GitHub.

Tas veidots, adaptējot Somijas Zviedru literatūras biedrības (Svenska litteratursällskapet i Finland – SLS) izstrādāto rīku aBBi.

aiRA palīdz ģenerēt attēlu alternatīvos tekstus (alt text) un transkripcijas, izmantojot mākslīgo intelektu (AI). Tāpat rīks spēj kodēt transkripcijas TEI XML formātā, kas paredzēts arhivēšanas vai pētniecības vajadzībām.

Jūs izmantojat savu personīgo OpenAI vai Google (Gemini) API atslēgu, un rīks darbojas pilnībā jūsu pārlūkprogrammā – nav iesaistīts nekāds aizmugursistēmas (backend) serveris. Attēli no jūsu pārlūkprogrammas tiek nosūtīti tieši izvēlētajam AI pakalpojumu sniedzējam, tiek saņemts rezultāts, un tas arī viss.

Tas nozīmē:

  • Jūs kontrolējat izmaksas. OpenAI vai Google piestāda rēķinu tieši jums, balstoties uz jūsu patēriņu un viņu standarta API tarifiem.
  • Jūs kontrolējat datus. Nekas neiet caur mūsu serveriem. Mēs neredzam ne jūsu attēlus, ne tekstu, ne API atslēgu.
  • Jūs izvēlaties pakalpojumu sniedzēju. Izvēlieties to AI pakalpojumu, kuram uzticaties un kuram jums ir pieejama atslēga.

Biežāk uzdotie jautājumi

Ko tieši dara aiRA?

Jūs augšupielādējat vai izvēlaties attēlu, un aiRA nosūta to AI modelim (OpenAI vai Google, atkarībā no jūsu konfigurācijas) kopā ar norādēm: vai nu aprakstīt attēlu (alternatīvajam tekstam), vai transkribēt tajā redzamo tekstu. Ja veicat transkripciju, varat iestatīt, lai rezultāts tiktu izvadīts TEI XML formātā.

Vai man ir nepieciešams profils, lai lietotu aiRA?

Mūsu sistēmā profils nav vajadzīgs, taču jums ir nepieciešama OpenAI vai Google API atslēga, jo aiRA izmanto šo uzņēmumu AI modeļus, lai veiktu faktisko attēlu analīzi.

Kā es varu iegūt API atslēgu?

Uzņēmumam OpenAI:

  1. Dodieties uz platformu platform.openai.com un reģistrējieties vai piesakieties
  2. Atveriet sava profila iestatījumu sadaļu "API keys".
  3. Izveidojiet jaunu slepeno atslēgu (secret key) un nokopējiet to drošā vietā – vēlreiz to apskatīt vairs nevarēsiet.
  4. Pievienojiet maksājumu informāciju, jo API izmantošana ir maksas pakalpojums (tas ir nodalīts no ChatGPT Plus abonementa).
  5. Iekopējiet atslēgu aiRA iestatījumos.

Uzņēmumam Google (Gemini):

  1. Dodieties uz Google AI Studio (aistudio.google.com).
  2. Piesakieties ar savu Google kontu.
  3. Noklikšķiniet uz "Get API key" un izveidojiet to.
  4. Iekopējiet atslēgu aiRA iestatījumos.

Vai rīka lietošana ir bezmaksas?

Pati aiRA ir bezmaksas. Taču aprakstu un transkripciju ģenerēšanai tiek izmantota jūsu API atslēga, un AI pakalpojumu sniedzēji par šo apjomu iekasē maksu. Parasti par attēlu tie ir daži centi vai summa, kas ir mazāka par centu, taču, strādājot ar lielu apjomu, izmaksas summējas. Pirms intensīvas lietošanas iepazīstieties ar OpenAI vai Google aktuālajiem cenrāžiem.

Kas notiek ar maniem attēliem un API atslēgu? Vai kaut kas tiek saglabāts?

Nekas netiek ne saglabāts, ne sūtīts mums. aiRA darbojas pilnībā jūsu pārlūkprogrammā:

  • jūsu API atslēga tiek saglabāta tikai jūsu pārlūkprogrammā (lokāli), lai tā nebūtu jāievada katru reizi no jauna. Mums tā nekad netiek pārsūtīta;
  • jūsu attēli tiek sūtīti tieši no jūsu pārlūkprogrammas uz OpenAI vai Google API – tie nekad nenonāk uz mūsu serveriem;
  • mēs neveicam nekādu žurnalēšanu (logging), neievācam analītiku par jūsu attēliem un nesaglabājam jūsu rezultātu kopijas. Mēs tos neredzam.

Īsāk sakot: jūsu datu attiecības ir ar OpenAI vai Google, nevis ar mums. Skatiet viņu attiecīgās privātuma politikas, lai uzzinātu, kā viņi apstrādā datus, kas nosūtīti uz viņu API (piemēram, vai dati tiek izmantoti modeļu apmācībai – tas atšķiras atkarībā no pakalpojumu sniedzēja un konta veida, tāpēc pārbaudiet viņu aktuālos noteikumus).

Vai es varu šo rīku izmantot ar jutīgiem vai privātiem attēliem?

Esiet piesardzīgi. Lai gan pati aiRA neko nesaglabā, jūsu attēli tomēr pamet jūsu ierīci un nonāk pie trešās puses AI pakalpojumu sniedzēja. Ja strādājat ar sensitīviem, personīgiem vai ierobežotas pieejamības materiāliem, pārskatiet OpenAI un Google datu izmantošanas un glabāšanas politikas, kā arī ievērojiet savas organizācijas datu apstrādes vadlīnijas.

Kādus AI modeļus es varu izmantot?

Vairākus OpenAI vai Google modeli, kuriem ir "vision" (attēlu izpratnes) kapacitāte. Precīzs saraksts ir konfigurējams un laika gaitā var tikt atjaunināts – pieejamās opcijas meklējiet iestatījumu panelī.

Vai es varu pielāgot to, kā tiek rakstīti apraksti?

Jā. Ir iespējams regulēt mākslīgā intelekta domāšanas līmeni un temperatūru jeb radošumu, kas ietekmē aprakstu valodu, kā arī ir iespējams pēc aprakstu ģenerēšanas tos rediģēt un pārtulkot uz citām valodām.

Kādās valodās ir iespējams atšifrēt tekstu?

aiRA atšifrēt spēj tekstus, kuri sastāv no latīņu alfabēta, tai skaitā burtus ar diakritiskām zīmēm, piemēram, “ā”, “ģ”, “ø”.

Vai aiRA darbojas mobilajās ierīcēs?

Jā, tā ir adaptīva tīmekļa lietotne (responsive web app). Tiesa, lielāku attēlu apjomu augšupielādei un pārskatīšanai ērtāk būs izmantot lielāku ekrānu.
